Dashboard Transportation Cost Variance - Regional View

───────────────────────────────────────────────────────────────────────────────
👤 USER STORY (Format: As a [role], I want [action], So that [benefit])
───────────────────────────────────────────────────────────────────────────────

As a **Regional Finance Manager**,
I want to **visualize transportation cost variance per country compared to budget**,
So that I can **identify cost overruns early and take corrective actions**.

───────────────────────────────────────────────────────────────────────────────
📊 BUSINESS CONTEXT (De ce este important? Care e impactul?)
───────────────────────────────────────────────────────────────────────────────

PROBLEMA:
  • Costurile de transport în Q1 2026 au depășit bugetul cu 12% în regiunea Sud-Est
  • Finance team nu are vizibilitate real-time pe deviații la nivel de țară
  • Procesul actual: export manual din SAP + Excel pivots (2 zile de lucru)

VALOARE AȘTEPTATĂ:
  • Reducere timp analiză: de la 2 zile → 15 minute
  • Identificare early warning: cost overruns detectate în aceeași zi
  • ROI: economii estimate €50K/an prin acțiuni corective rapide

STAKEHOLDER:
  • Requestor: Roxana Ene (Finance Manager)
  • Approver: Vasile Iordache (CFO)
  • End Users: Regional Finance Managers (5 persoane)

───────────────────────────────────────────────────────────────────────────────
✅ ACCEPTANCE CRITERIA (Format: Given/When/Then - Gherkin)
───────────────────────────────────────────────────────────────────────────────

AC1: Vizualizare variance cost per țară
  GIVEN I am a Finance Manager
  WHEN I open the Transportation Cost Dashboard
  THEN I see a bar chart showing Actual vs Budget cost per country for current quarter

AC2: Drill-down pe categorie cost
  GIVEN I click on a country bar in the chart
  WHEN the detail view opens
  THEN I see cost breakdown by: Fuel, Labor, Maintenance, Other
  AND I see variance % for each category

AC3: Filtru temporal
  GIVEN I am on the dashboard
  WHEN I select a time period (Month/Quarter/YTD)
  THEN the chart updates with data for selected period
  AND budget comparison is recalculated

AC4: Export pentru management
  GIVEN I need to present data in a meeting
  WHEN I click "Export to Excel"
  THEN I receive a formatted Excel file with:
    - Summary table (Country, Actual, Budget, Variance %)
    - Chart image embedded
    - Data refreshed as of [current date]

AC5: Performance
  GIVEN the dashboard loads
  WHEN I apply any filter
  THEN response time is < 3 seconds for data refresh

───────────────────────────────────────────────────────────────────────────────
📂 DATA REQUIREMENTS (Surse, granularitate, refresh)
───────────────────────────────────────────────────────────────────────────────

DATA SOURCES:
  • SAP FI-CO: tabla BSEG (actual costs)
  • Budget Excel: "2026_Budget_Transport.xlsx" (SharePoint)
  • Exchange Rates: ECB API (pentru conversie EUR)

GRANULARITATE:
  • Temporal: Daily actuals, Monthly budget
  • Geografic: Country level (8 țări: RO, BG, HU, GR, CZ, SK, PL, HR)
  • Categorie: Fuel, Labor, Maintenance, Other

DATA REFRESH:
  • Actuals: Daily at 06:00 AM (incremental load)
  • Budget: Manual upload la începutul fiecărui an

VOLUMETRIE:
  • ~500K records/month (actuals)
  • Historical data: 24 months rolling window
